Вступление

Woonder - CSS библиотека, предназначенная для построения гибких сеток. Woonder полностью построена на технологии Flexbox в соответствии с методологией BEM (block, element, modifier).

1. Basics

Для построения сетки с помощью Woonder необходимо выполнить несколько простых шагов:

  • 1. Добавить сетку <div class="grid"></div>
  • 2. Затем в сетке добавить строку <div class="grid__row"></div>
  • 3. И в конце нужно добавить в строке столько столбцов, сколько требуется <div class="grid__col"></div>

И все! Простая сетка с растягивающимися столбцами готова! Ведь просто!

2. Размеры колонок и размеры экранов

Woonder использует 12 колончатую систему построения сетки. Сетка и утилиты Woonder используют набор из шести контрольных точек.

Для того чтобы гибко настроить отображение колонок на различных экранах Вам к колонке просто нужно будет добавить класс-модификатор <div class="grid__сol grid__col_size_xs-12 grid__col_size_lg-6"></div>

3. Промежутки

В Woonder у Вас есть большие возможности по настройке промежутков между строками и колонками сетки.

Для того чтобы добавить промежутки вы должны использовать классы-модификаторы. Класс-модификатор может быть применен к сетке целиком <div class="grid grid_gap_0"></div> или к конкретной строке <div class="grid__row grid__row_gap_5"></div>.

Woonder использует 9 размеров промежутков сетки. По умолчанию ко всем колонкам сетки применены промежутки в 0.75rem с каждой стороны.
